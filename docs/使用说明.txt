Cubism Chat 使用说明
====================

适用版本：Live2D Cubism Editor 5.4 alpha1


一、启动

双击 Cubism Chat.bat 或 cubism-chat.exe，浏览器会自动打开。


二、授权（必须）

1. 打开 Cubism Editor 5.4
2. 菜单 → 外部应用程序集成设置
3. 找到 "Cubism Chat"，勾选权限复选框
4. 如果是首次使用，还需勾选"编辑"权限才能使用编辑功能


三、配置 API Key

1. 点击页面右上角齿轮图标
2. 填入 DeepSeek API Key
3. 模型名称默认为 deepseek-chat，可视需要修改
4. Cubism 端口默认为 22033，一般无需改动
5. 点击保存


四、使用

直接打字即可，AI 会自动理解你的意图并操作 Cubism Editor。

例：
  "查看模型参数结构"
  "把角度X的最大值改成90"
  "在角度Y的0.5位置添加一个键位"
  "为选中的所有对象创建旋转变形器"

输入框左侧 "/" 按钮可查看内置命令和自定义规则。


五、规则系统

规则是可复用的编辑模板，用 /规则名 快速调用。

预置规则：
  /three-points (/tp)  — 在参数最小/中间/最大值处加三个键位
  /list-model  (/lm)   — 查看模型完整结构

创建新规则：
  直接说 "添加规则：当你想要的操作描述"，AI 会生成规则草案让你确认。


六、注意事项

- 所有修改操作需在建模模式（Modeling）下进行
- 如遇报错，通常重启 Cubism Editor 后重试即可
- API Key 存储在浏览器本地，不会上传
